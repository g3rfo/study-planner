<h1>Study Planner</h1>
<p>
  A powerful and minimal <strong>React-based study planner</strong> built with <strong>Vite</strong> and <strong>Tailwind CSS</strong>.  
  Designed to help you organize lessons, subjects, and tasks — all stored locally in your browser.
</p>
<br>
<h2>Live Demo</h2>
<p>
  🔗 <a href="https://g3rfo.github.io/study-planner/#/"><strong>View on GitHub Pages</strong></a>
</p>
<br>
<h2>Features</h2>
<h3>1. Calendar Page</h3>
<ul>
  <li>Weekly schedule view (from 6:00 to 22:00)</li>
  <li>Scroll through weeks (3 before and 3 after current — total 7)</li>
  <br>
  <li>Add lessons with <code>Title</code>, <code>Link</code>(optional), <code>Date</code>, <code>Time range</code>(from / to), <code>Color</code>, <code>Notes</code>(optional)</li>
  <li>Open extended lesson view if it has a <code>Link</code> or <code>Notes</code></li>
  <li>Delete lessons easily</li>
  <br>
  <li>Save schedule templates based on selected weeks</li>
  <li>Delete or overwrite saved templates</li>
</ul>
<br>
<h3>2. Subjects Page</h3>
<ul>
  <li>Add new subjects with <code>Title</code> and <code>Color</code></li>
  <li>Rename subjects</li>
  <br>
  <li>Progress bar shows completed task ratio</li>
  <br>
  <li>Add tasks inside subjects</li>
  <li>Check and delete tasks</li>
</ul>
<br>
<h3>Appearance & Storage</h3>
<ul>
  <li>Toggle between light and dark themes</li>
  <li>All data stored locally via <strong>localStorage</strong></li>
</ul>
<br>
<h2>Tech Stack</h2>
<p><strong><u>React, Vite, Tailwind CSS, GitHub Pages</u></strong></p>
<br>
<h2>Getting Started</h2>
<ol>
  <li>
    Clone this repository:
    <pre><code>git clone https://github.com/&lt;yourusername&gt;/study-planner.git</code></pre>
  </li>
  <li>
    Navigate to the project folder:
    <pre><code>cd study-planner</code></pre>
  </li>
  <li>
    Install dependencies:
    <pre><code>npm install</code></pre>
  </li>
  <li>
    Start the development server:
    <pre><code>npm run dev</code></pre>
  </li>
</ol>
